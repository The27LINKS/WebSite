PDF Compress

Shrink PDF file size by stripping metadata, embedded thumbnails and unused objects.

Upload a PDF

Best results on PDFs with heavy metadata. Image-heavy PDFs may only shrink modestly — for those, export images first and convert back.

How it works

We re-save your PDF with object streams, strip title/author metadata and drop XFA attachments. Everything happens in the browser using pdf-lib.
